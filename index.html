Wellcome to Simple Start Page

A simple new tab page, customizable, without ads, trackers or heavy librarys

This extension is in development. There may be strange bugs.

Known issues

Background doesn't apply

The API used by Simple Start Page (localstorage API) has a storage limit

To save an image in localstorage, its necessary to convert the image to base64 string and save it in localstorage, and that doubles the image size

In future versions, I'll change the method to save files locally. For now, its recommended to use small images to avoid errors

Recommended: jpg, webp less than 2mb. You can still use images png more than 2mb, but it can cause errors.

How do I edit my bookmarks

Now, bookmarks are saved in your bookmarks folder in your browser. You can edit or reorder bookmarks in browser's builtin bookmark manager

In future updates, it will be possible to edit bookmarks inside SSP page
